1. PROJECT OVERVIEW

The SME Food App is a comprehensive mobile application developed to streamline the food ordering process for customers, connecting them directly with restaurants within the SME Food ecosystem. Built using React Native, Node.js, and MongoDB, this cross-platform app enables users to browse restaurant menus, place orders, track order status in real-time, and provide feedback, thereby enhancing the overall dining experience. My role involved leading the development of both the front-end and backend functionalities of the app.

2.MY ROLE

As a Full-Stack Developer, my responsibilities encompassed the entire lifecycle of the SME Food App, from conceptualization to deployment. Key contributions include:

  • Designing and implementing the mobile app's UI/UX using React Native, ensuring a seamless and intuitive user experience across Android platforms.
  • Developing a scalable backend system with Node.js and Express, capable of handling high volumes of orders and processing real-time data efficiently.
  • Integrating MongoDB for robust database management, storing user profiles, order histories, restaurant menus, and feedback.
  • Implementing secure authentication and authorization mechanisms using JWT (JSON Web Tokens) to protect user data and ensure privacy.
  • Creating RESTful APIs for communication between the mobile app and the backend server, enabling real-time order tracking and notifications.

3. TECHNICAL HIGHLIGHTS

These are the technical highlights  

  • Leveraged React Native for cross-platform mobile app development, reducing development time and costs while maintaining high performance.
  • Utilized Node.js and Express for backend development, providing a scalable and efficient server-side solution.
  • Employed MongoDB for database management, offering flexibility and scalability for storing and retrieving application data.
  • Implemented JWT for secure authentication, safeguarding user accounts and personal information.
  • Integrated API for order status updates, enhancing user engagement and satisfaction.

4. OUTCOMES

These are the project outcomes  

  • Successfully launched the SME Food App on Android platforms, reaching a broad audience of users.
  • Streamlined the food ordering process, significantly improving customer satisfaction through real-time order tracking and notifications.
  • Enhanced operational efficiency for restaurants by integrating order processing directly into their administrative systems.
